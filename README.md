Сайт для заказа такси с водителями-женщинами. Проект создается для обеспечения комфорта и безопасности пассажирок

- **ASP.NET Core 8.0** (Razor Pages)
- **Entity Framework Core** (работа с БД)
- **Individual Accounts** (аутентификация и регистрация)
- **Bootstrap 5** (стилизация)
- **Git & GitHub** (контроль версий)
